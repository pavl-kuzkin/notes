# Trend Mint Laptop Setup

### Chrome
download the .deb and then use 
```shell
sudo apt install ./<file>.deb
```

### zsh

Install zsh 
```shell
apt install zsh
```

Make zsh default shell
```shell
chsh -s $(which zsh)
```
